🎯 The Delegation Decision

Before delegating any task to an agent, run it through this framework. The goal: delegate the right things, keep control of the rest.

Core FrameworkThe 2×2 Matrix

Agent CAN do itAgent CANNOT do it
Low risk✅ DelegateHuman does it
High risk⚠️ Delegate + review❌ Don't delegate

Can the agent do it?

Is the task well-defined?
Can the output be verified?
Does it need post-cutoff knowledge?
Does it require YOUR specific context?

How risky is it?

Low: Summarize, format, search
Medium: Write code, create content
High: Delete files, send emails
Critical: Irreversible, personal data, public

PatternsFive Ways to Delegate

1. Fire and Forget ✅

Low risk, agent is competent. Example: "Summarize this article." Human involvement: none.

2. Draft and Review ⚠️

Medium risk or uncertain quality. Example: "Write a blog post about X." Human reviews before publishing.

3. Plan and Approve

High risk, but agent can plan. Example: "Reorganize the file structure." Agent proposes → human approves → agent executes.

4. Supervised Execution

High risk, step by step. Example: "Deploy the new version." Human approves each step individually.

5. Human Only ❌

Critical risk or requires human judgment. Example: "Should we hire this person?" Agent can gather info, but human decides.

PitfallsCommon Mistakes

🔴 Over-delegation

"Let the agent handle everything." Invisible errors accumulate.

Fix: Start with Draft+Review for everything. Promote to Fire+Forget after proving reliability.

🔴 Under-delegation

"I'll do it myself, the agent makes mistakes." You're paying for an agent you don't use.

Fix: Start with low-risk tasks. Build trust gradually.

🔴 No Verification

"The agent said it's done, so it's done." The biggest source of failures.

Fix: Every high-risk task needs a verification step. Not "did it run?" but "is the result correct?"

Progressive TrustThe Trust Ladder

Week 1   🟢 Everything needs approval (driving instructor)
Week 2   🟢 Routine tasks auto-approved (familiar roads)
Week 4   🟡 Research runs autonomously (highway driving)
Week 8   🟠 Complex workflows delegated (city driving)
Month 3 🟠 Sub-agents run scheduled tasks (solo driving)
Month 6 🔴 System monitors itself (dashcam)

Don't skip levels. Each level builds on trust earned at the previous level. Jumping from Week 1 to Month 3 is how agents break things.
