🎯 The Delegation Decision

Before delegating any task to an agent, run it through this framework. The goal: delegate what agents do well, keep what requires human judgment.

Core FrameworkThe 2×2 Matrix

Agent CAN do itAgent CANNOT do it
Low risk✅ DelegateHuman does it
High risk⚠️ Delegate + review❌ Don't delegate

Can the agent do it?

Is the task well-defined?
Can the output be verified?
Does it need post-cutoff knowledge?
Does it need YOUR specific context?

What's the risk?

Low: Summarize, format, search
Medium: Write code, create content
High: Delete files, send emails
Critical: Irreversible, personal data, public

Delegation PatternsFive levels of human involvement

Pattern 1: Fire and Forget ✅

Low risk, agent is competent. Example: "Summarize this article." Human involvement: none.

Pattern 2: Draft and Review ⚠️

Medium risk or uncertain quality. Example: "Write a blog post." Human reviews before publishing.

Pattern 3: Plan and Approve

High risk, but agent can plan. Example: "Reorganize the files." Agent proposes → human approves → agent executes.

Pattern 4: Supervised Execution

High risk, step by step. Example: "Deploy the new version." Human approves each step individually.

Pattern 5: Human Only ❌

Critical risk or requires judgment. Example: "Should we hire this person?" Agent gathers info, human decides.

Common MistakesWhat goes wrong

🔴 Over-delegation

"Let the agent handle everything." Invisible errors accumulate. Fix: Start with Pattern 2 for everything. Earn Pattern 1.

🔴 Under-delegation

"I'll do everything myself." You're paying for an agent you don't use. Fix: Start with low-risk tasks. Build trust.

🔴 No Verification

"The agent said it's done, so it's done." Biggest source of failures. Fix: Every high-risk task needs "is the result correct?" not "did it run?"

Building TrustThe Trust Ladder

Week 1: Everything needs approval (driving instructor)
Week 2: Routine tasks auto-approved (familiar roads)
Week 4: Research runs autonomously (highway driving)
Week 8: Complex workflows delegated (city driving)
Month 3: Sub-agents run scheduled tasks (solo driving)
Month 6: System monitors itself (dashcam review)

Don't skip levels. Each builds on trust earned at the previous level.
