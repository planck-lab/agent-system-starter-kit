# Skill: Deep Research

## Trigger
"Research...", "Deep dive into...", "What's the current state of..."

## Method
Follow the scientific method:
1. **Define** the research question clearly
2. **Search** using web search (3-5 diverse sources)
3. **Evaluate** source quality (peer-reviewed > industry > blog)
4. **Synthesize** findings into structured output
5. **Cite** every claim with source URL

## Output Format
```markdown
# [Research Topic]

## Key Findings
- Finding 1 (Source: URL)
- Finding 2 (Source: URL)

## Analysis
[Your synthesis — what do the findings mean together?]

## Open Questions
[What couldn't be answered? What needs more research?]

## Sources
1. [Author (Year)] — Title. URL. [Tier A/B/C]
```

## Source Tiers
- **Tier A:** Peer-reviewed (conferences, journals)
- **Tier B:** arXiv/preprints with significant citations
- **Tier C:** Industry reports, official documentation

## Rules
- Minimum 3 sources per research topic
- Always note when a claim has no source ("Estimate", "No source")
- If sources contradict each other: report BOTH perspectives
- Save output to Vault (Research/ folder) automatically
